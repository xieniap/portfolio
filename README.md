## Featured Project: CloudVault

CloudVault is a production-ready demonstration of automated infrastructure designed for secure, cost-effective training platforms.

- **Infrastructure:** AWS (EC2, S3, VPC, IAM)
- **IaC:** Terraform for automated environment provisioning.
- **CI/CD:** Automated deployment pipelines via GitHub Actions.
- **Security:** Strict IAM 'Least-Privilege' boundaries and encryption at rest aligned with HIPAA compliance requirements.
- **FinOps:** Implementation of burstable compute instances and S3 Lifecycle policies resulting in a ~35% reduction in cloud overhead.
